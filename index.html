Twitter Muting Tool

This tool creates exhaustive lists of variations on a given phrase in order to fully circumvent methods of censoring them on Twitter so you can mute them entirely. For example, a word like "heck" could be censored in various ways:

  • h3ck
  • h*ck
  • h**k
  • h***
  • he(k
etc. Enter a phrase below and the tool will generate as many of these variations for that phrase as possible.
